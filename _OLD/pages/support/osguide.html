Know your OS

Welcome to our guide on identifying which operating system your device is running.

Apple® Products

Identifying the operating system of an Apple® product is the easiest. Apple® products can be distinguished by the iconic logo (an apple with a bite taken out of it) displayed on the back of all devices. All Apple® products will run on one of three operating systems: macOS®, iOS or iPadOS®.

If you are using a Mac® or MacBook® (Apple's desktop and laptop computers respectively), you will be running macOS®.

If you are using an iPad® (Apple's tablet) which has received a software update since September 2019, your device will be running iPadOS®.

Finally, if you are using an iPhone® (Apple's smartphone), or an iPad® (Apple's tablet) which has not received a software update since September of 2019, your device will be running iOS.

Smartphones and Tablets

If you are using a smartphone not made by Apple®, it is most likely running Android™. Additionally, if you are using a tablet not created by Apple®, it is likely to be running Android™.

Desktops and Laptops

The majority of desktop and laptop computers will be running Windows, although there may be some exceptions. Windows devices can be identified by the four-square symbol on the taskbar.

If you have a laptop, cannot see the four-square Windows logo, and have not changed the operating system since purchase, your device is likely to be running ChromeOS.

Some dekstop and laptop computers may be running a version of Linux. In this case you will need to identify the version of Linux you are using. Most Linux distributions will display their name and logo when starting the device.
